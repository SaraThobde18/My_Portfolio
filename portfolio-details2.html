Project Details

Disease Prediction

This project utilizes machine learning to predict diseases based on user-input symptoms. It uses a trained Random Forest Classifier on a medical dataset to analyze symptom patterns and determine possible illnesses. A Flask-based web application allows users to select symptoms and receive instant predictions. The system enhances early diagnosis and decision-making in healthcare.
